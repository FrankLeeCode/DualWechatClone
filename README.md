# DualWechatClone

<img src="/res/AppIcon.png" width="100"/>

> 一键式解决MacOS微信双开问题

## 双开效果

![双开效果](/res/illustration.png)

## 使用方法
1. 双击app打开
2. 输入管理员密码
3. 等待克隆的微信弹出
4. 将克隆的微信固定到Dock（程序坞）方便打开
5. 【可选】更改克隆的微信图标方便区分

## 可用版本：
- 4.0.6.240 (2025-8-10)
